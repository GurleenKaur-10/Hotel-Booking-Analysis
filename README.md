# Hotel-Booking-Analysis
Analyzed hotel booking datasets to uncover cancellation patterns, assess revenue impact, and propose strategies such as pricing adjustments, seasonal promotion planning, and improved customer targeting to reduce cancellations. I used Python for data cleaning and Power BI to create a stakeholder dashboard.
